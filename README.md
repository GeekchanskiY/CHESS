# CHESS

Chess game using pygame

Some advantages:

<ul>
  <li>King movement</li>
  <li>Queen movement</li>
  <li>Rook movement</li>
  <li>Bishop movement</li>
  <li>Knight movement</li>
  <li>Some draw rules</li>
</ul>
